## bashmarks

shell的书签，因为总是cd浪费时间，所以添加书签更方便

启动：

```
source ~/.local/bin/bashmarks.sh
```

编辑：

```
vi ~/.local/bin/bashmarks.sh
```



## tmux

终端复用神器

启动：

```
tmux
```

进入：

```
tmux attach -t 0
```

关闭窗口：

```
control b + &
```

退出但是不关闭：

```
control b + d
```

- https://linuxtoy.org/archives/scripting-tmux.html
- http://mingxinglai.com/cn/2012/09/tmux/
