## 参考教程

在使用某个命令前，如果忘记了这个命令的使用方法，可以使用help来查看一些说明

```
ls --help
```

**安装和卸载**

```
sudo apt-get install xxx
sudo apt-get remove xxx
```

要想执行sh文件：

```
sudo chmod 755 run.sh
然后使用 ./run.sh
或者使用bash run.sh执行
```
